Overview

Welcome to the Manhattan & Associates Design System (MANH-DS), a responsive and multipurpose service providing templated user-experience/user-interaction elements for a broad range of applications -including but certainly not limited too- custom panels, administrative dashboards, CMS, CRM, SAAS, etc.

MANH-DS has a sleek, clean and intuitive flat style design setup to make your next project look awesome and user friendly. MANH-DS also touts a robust collection of plugins and UI components working seamlessly on all major web browsers, tablets and phones.

We put a lot of love and effort to make MANH-DS a useful tool for everyone, and are keen to release continuous updates with new features in our future releases.

Styles

Heads up!
We use 3 CSS files in this theme: a style file, ui file and theme file. All CSS files are created via Less version. So you can easily modify style with less version or directly in style file if you don't want to use LESS.

Stylesheets

Here's a list of the stylesheet files we are using with this template, you can find more information opening each file:

File Name Description
style.css The main CSS file of the template. You will find all general style for sidebar, topbar and page content.
ui.css This file content all user interface style, such as button, form, table, modals...
theme.css Theme file define all color variations. There are 7 main colors with 4 variations for each color, so 28 variations in total.
